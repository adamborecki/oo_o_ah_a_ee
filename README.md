# oo-oh-a-ee
a webapp by adam borecki with help from Claude code

## about
use this tool to practice listening to the frequency spectrum and imagining that you're hearing different vowels.

## practice
practice listening by reorganizing waveforms.

try it out:

- https://adamborecki.github.io/oo-o-ah-a-ee/

You need:
- a WAV file in 6 segments:
-- segment 1 is dry/reference
-- segments 2-6 are the same as 1 except there's an EQ boost around 250 Hz, 500 Hz, 1 kHz, 2 kHz, and 4 kHz in random order
